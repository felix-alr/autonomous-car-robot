# HS-AMR Example Android App
This app provides a example HMI for the HS-AMR Robot. 
Implemented are BT Connection/Dialog, Position request and external control.

# Required Software
- [Android studio Narwhal](https://developer.android.com/studio?hl=de)\
- Java SDK (should be automatically installed when installing Android Studio)

# Setup
- Open Project in Android studio (File --> Open)
- wait until Android Studio has finished the import
- Build / Sync Android Studio with Project (Should be done Automatically by Gradle)
- Plug in Tablet with enabled USB-Debugging
- Select Tablet as running device
- Press "Run APP"
